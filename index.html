Generador de Base de Datos

El siguiente link generará una base de datos de 5000 clientes.

Se incluirán datos tales como: ID, Rut, Nombre, Apellido, Segundo Apellido, Fecha de Nacimiento, Dirección, País, Ciudad, Código Postal, Teléfono, Correo electrónico, Ocupación, Estado Civil, Centro de Costo y Compras anuales.

Se descargará un archivo en formato .csv el cuál se debe abrir en block de notas, importar desde Excel o importar directamente en PowerBI.
